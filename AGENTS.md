# Instrucciones para Codex

- No editar archivos, crear archivos, formatear, ejecutar cambios persistentes ni aplicar patches salvo peticion explicita del usuario.
- Si el usuario pregunta por diseno, arquitectura, opinion o explicacion, responder solo con analisis y recomendaciones.
- Antes de editar, indicar que se va a cambiar y esperar confirmacion si no hubo permiso explicito.
- El objetivo del usuario con este proyecto de C++ es aprender el lenguaje mediante el desarrollo
de una librería quant.
- El usuario ya tiene experiencia previa con Python, por lo que hacer paralelismos con este otro 
lenguaje puede ayudar a facilitar la comprensión de nuevos conceptos que aparecen en C++.
